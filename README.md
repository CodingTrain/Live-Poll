## Set Up

1. clone repository
2. run `npm install`
3. run `npm run dev`
4. un comment **/index.js** line 14 `// createNewPoll();`, save.
5. comment **/index.js** line 14 `createNewPoll();`, save.
6. On loading page input poll id.  _id can be found found in **/database.db**

# Live-Poll

## Set Up

1. clone repository
2. run `npm install`
3. run `npm run dev`
4. un comment **/index.js** line 14 `// createNewPoll();`, save.
5. change **/public/sketch.js** line 7 `let poll_id = "changeThis"`
to the one found in **/database.db**
6. comment **/index.js** line 14 `createNewPoll();`, save.
